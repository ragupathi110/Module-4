## DICTIONARY - CHECK WHETHER DICTIONARY OR NOT


### AIM  
To write a Python program check whether a given key exists in a dictionary or not.

### ALGORITHM

1.Start

2.Take the dictionary dict_obj and the key key_to_check as input

3.Use the in operator to check:

4.If key_to_check in dict_obj

5.If true, return True

6.Else, return False

7.End

### PROGRAM

dic={'0':1,'1':2,'2':3}    <br />
x=input()       <br />
if x in dic.keys():     <br />
    print("True")    <br />
else:   <br />
    print("False")
